# HMEL Tech Conference Frontend

Одностраничный лендинг конференции HMEL, созданный на Vite + React с фирменным стилем и адаптивной вёрсткой.

## Основные секции

- **Герой-блок** с названием конференции, датами и CTA.
- **О событии** с целями и задачами мероприятия.
- **Расписание** с табами по дням и залам.
- **Партнёры** с логотипами в фирменных градиентах.
- **Карта площадки** с встраиваемым виджетом Яндекс.Карт.
- **FAQ** с аккордеонами.
- **Футер** с контактами и ссылками.

## Быстрый старт

```bash
npm install
npm run dev
```

После запуска dev-сервер доступен по адресу [http://localhost:5173](http://localhost:5173). Для просмотра на других устройствах используйте локальную сеть и параметр `--host`.

## Сборка и предпросмотр

```bash
npm run build
npm run preview
```

Команда `build` создаёт production-сборку в каталоге `dist/`, а `preview` запускает локальный сервер для проверки результата.

## Структура проекта

```
public/
  assets/         # Логотипы партнёров, фавикон и декоративные SVG
src/
  components/     # Переиспользуемые UI-компоненты (аккордеон)
  sections/       # Секции лендинга
  index.css       # Глобальные стили и адаптивность
  main.tsx        # Точка входа приложения
```

## Лицензия

Проект предназначен для демонстрационных целей в рамках задания и не подразумевает публичного распространения.
